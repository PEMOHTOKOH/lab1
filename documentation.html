Документація до програми (Лабораторна робота №1)

Ця документація описує модульний Java-додаток, який моделює банківські операції.

1. Опис програми

Програма реалізує банківські операції, такі як:

  • Створення банківських карток для користувачів.
  • Перевірка віку користувача.
  • Управління підписками.

Додаток побудований із використанням модульної системи Java (Jigsaw).

2. Структура програми

Програма складається з кількох модулів:

  • jmp-dto: Містить DTO-класи (User, BankCard, Subscription).
  • jmp-bank-api: Інтерфейс Bank.
  • jmp-service-api: Інтерфейс Service.
  • jmp-cloud-bank-impl: Реалізація BankImpl.
  • jmp-cloud-service-impl: Реалізація ServiceImpl.
  • jmp-main: Головний клас із запуском.

3. Інструкції з використання

  1. Скомпілюйте проект командою: mvn clean install.
  2. Запустіть програму командою: java -p out -m jmp.main/edu.hneu.mjt.andrii.Main.
  3. Перевірте консольний вивід, який показує:
    • Створення карти з номером (наприклад, 1234-5678-9012-3456).
    • Результати перевірки віку (true/false).
    • Ідентифікатор підписки (наприклад, SUB1).

4. Примітки

- Проект використовує лише стандартні засоби Java без зовнішніх бібліотек. - Для коректної роботи необхідно налаштувати модульні залежності в `module-info.java`. - Джерельний код доступний за посиланням: [https://github.com/YourUsername/lab1-java-modules](https://github.com/YourUsername/lab1-java-modules).
